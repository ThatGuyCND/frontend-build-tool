## Step 1: Installation Requirements
> "... cut a hole in a box ..."

* :beer: Homebrew
```bash
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
* Node
```bash
brew install node
```
* Gulp
```bash
npm install -g gulp
```
* :bird: Bower
```bash
npm install -g bower
```
